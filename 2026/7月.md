### CIDR とは

CIDR（Classless Inter-Domain Routing）は、IP アドレスの範囲を `192.168.1.0/24` のように表す書き方。

`/24` の部分は「先頭から何ビットをネットワーク部として見るか」を表す。IPv4 は全部で 32 ビットなので、`/24` なら残り 8 ビットがホスト部になり、`192.168.1.0` 〜 `192.168.1.255` の範囲を指す。

よく見る例:

- `/32`: IP アドレス 1 個だけ
- `/24`: 256 個分の範囲
- `/16`: 65,536 個分の範囲

ファイアウォールや VPC の設定では、「どの IP 範囲を許可するか」を指定するためによく使う。

## 💻 ターミナル環境構築メモ (2026-07)

手元の開発環境をクリーンに保ちつつ、足腰を強くするために導入したツールと設定のまとめ。

---

### 🚀 1. fd (純正ファイル検索ツール)

`find` コマンドのRust製上位互換。`.git` や `node_modules` を自動で無視して、ファイル名で爆速検索する。

#### 📥 インストール
$ brew install fd

#### 💡 よく使う純正コマンド
* 基本検索:
$ fd ghostty

#### ⚠️ パスにスペース（空白）がある場合の注意点
検索結果のパスに空白がある場合、そのまま `cat` や `nvim` などのコマンドに渡すと「別の引数」とみなされてエラーになる。必ずパス全体を `""`（ダブルクォーテーション）で囲むこと。

* NGな例:
$ cat Library/Application Support/com.mitchellh.ghostty/config.ghostty

* OKな例:
$ cat "Library/Application Support/com.mitchellh.ghostty/config.ghostty"

---

### 🛠️ 2. Ghostty (ターミナルエミュレータ)

高速でモダンな最新のターミナル。

#### ⚙️ クイックターミナル設定 (cmd+space 割り当て)
画面上部からシュッとターミナルを出し入れできる設定。

* 設定ファイルのパス:
`~/.config/ghostty/config`（または `Library/Application Support/com.mitchellh.ghostty/config.ghostty`）

* 追記した設定:
keybind = global:cmd+space=toggle_quick_terminal

#### ⚠️ Mac標準のSpotlightとの衝突回避
`cmd+space` はMacのSpotlightと競合するため、以下の手順でMac側の機能をオフにする。

1. Macの「システム設定」を開く。
2. 「キーボード」 ➔ 「キーボードショートカット...」を開く。
3. 左メニューの 「Spotlight」 を選ぶ。
4. 「Spotlight検索を表示」 のチェックボックスを外す。

### Zellij × Ghostty 開発環境設定まとめ

#### 1. Zellij の概要
Zellijは、Rust製で開発されている非常にモダンで高速な**ターミナルマルチプレクサ（画面分割・管理ツール）**です。tmuxの現代版にあたります。

* **直感的なUI**: 画面下部に常に操作ガイドが表示されるため、エンジニアがキーバインド迷子になりません。
* **強力なレイアウト（KDL）**: 独自の記述フォーマットで、起動時の画面分割状態や実行コマンド（`ping` や `nvim` など）をあらかじめ定義しておけます。
* **デフォルトでのマウス対応**: スクロールやペインのリサイズが標準で直感的に行えます。

---

#### 2. 現在の画面レイアウト構造
定義したカスタムレイアウトにより、起動時に以下の配置が自動生成され、最初から左側のメイン作業領域にフォーカス（入力可能状態）が当たります。

```text
+-------------------------------------------------------------+
| [Tab Bar]                                                   |
+------------------------------+------------------------------+
|                              | (A) ping 127.0.0.1           |
|                              +------------------------------+
|                              | (B) terminal                 |
|                              +------------------------------+
| [Main] (50%)                 | (C) terminal                 |
|                              +------------------------------+
|                              | (D) terminal                 |
|                              +------------------------------+
|                              | (E) terminal                 |
+------------------------------+------------------------------+
| [Status Bar]                                                |
+-------------------------------------------------------------+
```

---

#### 3. 確定したキーバインド運用ルール

Macの `Option` キーを `Alt` メタキーとしてGhosttyに認識させ、競合を排除したことで、**「モード切り替え（Ctrl + p等）」を一切挟まない一発移動**が実現しています。

| 操作対象 | アクション | ショートカットキー | 挙動のレイヤー |
| :--- | :--- | :--- | :--- |
| **ペイン（分割画面）** | **左右**の移動 | `Option` + `←` / `→` | **Zellij** が処理 |
| | **上下**の移動 | `Option` + `↑` / `↓` （または `h/j/k/l`） | **Zellij** が処理 |
| **タブ（外枠）** | **左右タブ**の切り替え | `Option` + `Command` + `←` / `→` | **Ghostty** が処理 |

---

#### 4. 最終的な Ghostty 設定（`config`）

これまでの調整をすべて反映した `~/.config/ghostty/config` の関係部分の抜粋です。

```conf
# -------------------------------------------------------------------
# ウィンドウ・外観設定
# -------------------------------------------------------------------
background-opacity = 0.85
background-blur-radius = 30
mouse-hide-while-typing = true
scrollback-limit = 100000
mouse-shift-capture = true

# -------------------------------------------------------------------
# Mac特有のキーバインド調整（Alt/Optionマッピング）
# -------------------------------------------------------------------
# 左Optionキーを「Alt」として動作させ、Zellijの一発移動を有効化
macos-option-as-alt = left

# Ghostty独自のクイックターミナル
keybind = global:cmd+space=toggle_quick_terminal

# -------------------------------------------------------------------
# 画面分割（Ghosttyレイヤーでの分割・リサイズ用）
# -------------------------------------------------------------------
keybind = super+d=new_split:right
keybind = super+shift+d=new_split:down

keybind = super+ctrl+left=resize_split:left,10
keybind = super+ctrl+right=resize_split:right,10
keybind = super+ctrl+up=resize_split:up,10
keybind = super+ctrl+down=resize_split:down,10

# -------------------------------------------------------------------
# キーバインド競合回避 & タブ移動設定
# -------------------------------------------------------------------
# 1. Ghostty側のデフォルトのAlt+左右矢印を解除（Zellijのペイン移動へパススルーするため）
keybind = alt+left=unbind
keybind = alt+right=unbind

# 2. Ghostty自体のタブ移動を Option + Command + 左右矢印 に割り当て
keybind = super+alt+left=previous_tab
keybind = super+alt+right=next_tab
```
