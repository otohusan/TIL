### ruby uniq

配列から重複した要素を取り除いた新しい配列を返す

```ruby
p [1, 3, 2, 2, 3].uniq   # => [1, 3, 2]
```

ブロックを渡した場合は、ブロックが返した値が重複したものを削除

```ruby
p [1, 3, 2, "2", "3"].uniq { |n| n.to_s } # => [1, 3, 2]
```

要素を先頭から辿っていき、最初に出現したものが残る
