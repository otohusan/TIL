## 3 月 5 日
### Rail index_with
- 配列から「配列の要素をキー、ブロックの戻り値をバリュー」とするハッシュを簡単に作成できる
- `to_h` でも同様のことはできるが、`index_with` の方がより意図が明確で簡潔に書ける

```rb
users = ["alice", "bob", "charlie"]

# to_h を使う場合
users.map { |name| [name, name.length] }.to_h
# => {"alice"=>5, "bob"=>3, "charlie"=>7}

# index_with を使う場合（Rails 拡張）
users.index_with { |name| name.length }
# => {"alice"=>5, "bob"=>3, "charlie"=>7}

# 固定値を入れたい場合はブロックなしでも書ける
users.index_with(0)
# => {"alice"=>0, "bob"=>0, "charlie"=>0}
```

### Rails フォームオブジェクト
- 機能1つを特定のフォームオブジェクトに対応させる
  - 特定のケースのみ使用される機能でモデルが膨らむことや
  - いくつものモデルをコントローラーから呼び出すことなどを防ぐ
- バリデーションなども記述することで、追う必要があるファイルをはっきりさせられる

### load イベント, DOMContentLoaded イベント
#### load イベント
- ページ全体が読み込まれた時に使用される
#### DOMContentLoaded イベント
- ページの DOM の読み込みが完了すれば実行される
- 画像、サブフレーム、非同期スクリプトの読み込みの完了は待たない
