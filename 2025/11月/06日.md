## 太字に関して

太字として表示する方法には以下がある

- `strong` タグ
- `b` タグ
- `span` タグに `css`

UI 上の見た目は同じだけど、HTML としては使い分けるべきらしい

### strong

- その言葉の 重要性、深刻性、緊急性 を表す
- UI の見た目を変更したいだけの場合には利用しない

### b

- 文書の概要でのキーワード、レビューでの製品名に使われる
- 重要なものとしてマークせずにキーワードを強調するために使う
- 他により適切な要素がない場合に、最後の手段として使用すべきであり、見出しは h1 から h6、強調は em、重要性は strong、テキストのマークまたは強調は、mark を使うべきらしい

### span に css

- 単に太字として表示したい時
