# 2025 年 11 月 09 日

## Web 標準

- ウェブサイトを構築するために使用する技術
- 仕様書として存在し、この技術がどのように動作すべきかを詳細に記述されてる
- 標準化団体によって作成される
  - [HTML Living Standard](https://html.spec.whatwg.org/multipage/) (すべての HTML 要素とそれに関連する API など)
  - [WHATWG](http://spec.whactwg.org/) ( fetch や HTML, URL, WebSocket など)
  - [ECMA](https://ecma-international.org/) ( ECMAScript, C#, Dart など)
  - [W3C](https://www.w3.org/ja/about/) ( WASM など)

### ChatGPT にまとめてもらうと

#### 「Web 標準」とは？

Web プラットフォームで使われる技術の仕様（HTML/CSS/DOM/JS/各種 Web API/HTTP/URL/TLS など）を
公的な標準化団体が文書として定義し、かつ主要ブラウザで相互運用できる状態になっているもの。

### 余談

> https://www.publickey1.jp/blog/13/googledartecma.html

昔 Google が ブラウザに Dart VM を搭載して、 ブラウザ上で実行可能にしていたらしい、これが実現して主要ブラウザにて Dart が実行可能になると、 Dart を Web 標準 と呼べる可能性もあったぽい
