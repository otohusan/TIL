## ReturnType

「関数の型」から「戻り値の型」を取り出すことができる

```ts
function greet(name: string): string {
  return `Hello, ${name}`;
}
```

これの型は 以下の通り

```ts
typeof greet; // => (name: string) => string;
```

ReturnType を使えば 戻り値である **string** を取得する

```ts
ReturnType<typeof greet>; // => string
```

### 余談

引数の型を取得するのは `Parameters<T>`

```ts
Parameters<typeof greet>; // => [name: string]
```
