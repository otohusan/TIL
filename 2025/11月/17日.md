# 2025 年 11 月 17 日

## コントローラ名

コントローラー名を具体化、粒度を小さくすると

- RESTful な設計がしやすくなり、index・create・show・update・destroy など基本アクションだけで十分になる
- 単一責任の原則に近づき、役割が明確になる

## 特定ファイルだけ stash

```
git stash push -m "なんらかの名前" -- app/example1.ts app/example2.ts
```

新規作成ファイルの場合は以下のように -u をつける

```
git stash push -u -m "なんらかの名前" -- app/example1.ts app/example2.ts
```
