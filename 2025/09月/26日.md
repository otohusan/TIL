## Redash のクエリ (Mongo)

### 例

```json
{
  "collection": "コレクション名",
  "query": {
    // 値が一致するフィールドをとる
    "is_sales": true,
    // $ne は「ではない」を表せる
    "role": { "$ne": 1 }
  },
  "sort": [
    {
      "name": "_id",
      "direction": -1
    }
  ],
  "limit": 20
}
```

Aggregation を使えば複雑なクエリを取得できるらしい（勉強中）
