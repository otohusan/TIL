## Redash のクエリ (Mongo)

### 例

```json
{
  "collection": "コレクション名",
  "query": {
    // 値が一致するフィールドをとる
    "is_sales": true,
    // $ne は「ではない」を表せる
    "role": { "$ne": 1 }
  },
  "sort": [
    {
      "name": "_id",
      "direction": -1
    }
  ],
  "limit": 20
}
```

- $project は出力のフォーマットを調整できる
- $lookup は違うテーブルを参照できる
