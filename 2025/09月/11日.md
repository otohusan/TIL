## Apollo useQuery

option として skip という値を渡せる、渡せばクエリを行わずにスキップできる

```ts
function UserInfo({ shouldSkip, userId }) {
  const { data, loading } = useQuery(GET_USER, {
    variables: { id: userId },
    skip: shouldSkip, // trueならクエリを実行しない
  });
}
```
