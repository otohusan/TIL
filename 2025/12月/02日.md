## exactOptionalPropertyTypes

オプションプロパティに undefined の代入することを禁止するコンパイラオプション

標準だと

```ts
interface User {
  name: string;
  // オプショナルに指定
  nationality?: "India" | "China";
}

const user: User = {
  name: "hogehoge",
  // undefined を許容する
  nationality: undefined,
};
```

exactOptionalPropertyTypes を on にした場合

```ts
interface User {
  name: string;
  // オプショナルに指定
  nationality?: "India" | "China";
}

const user: User = {
  name: "hogehoge",
  // undefined は代入できない
  nationality: undefined,
};
```

## ?? と || の違い

- ?? は 左辺が undefined か null の場合に右辺を参照する
- || は 左辺が 何らかの偽値 である場合に右辺を参照する
