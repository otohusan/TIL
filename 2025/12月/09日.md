## rescue_from

Rails の`rescue_from`は、コントローラーで例外（エラー）が発生した時に、特定の処理を実行できる仕組み。
例えば、特定の例外が発生した時にエラーページを表示したり、ログを記録したりできる。

### 例

```rb
class ApplicationController < ActionController::Base
	rescue_from ActiveRecord::RecordNotFound, with: :render_404

	def render_404
		render file: "public/404.html", status: :not_found
	end
end
```

この場合、`ActiveRecord::RecordNotFound`が発生したら`render_404`メソッドが呼ばれる。
