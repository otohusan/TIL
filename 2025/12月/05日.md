## freeze_time

`freeze_time`は、テストなどで現在時刻を固定したいときに使うメソッドやヘルパー。
Rails のテストや RSpec でよく使われる。

### 例（RSpec の before ブロックで freeze_time を使う）

```rb
describe '日時を使うテスト' do
	before { freeze_time }

	it '現在時刻が固定されている' do
		expect(Time.now).to eq(Time.zone.now)
		# ここでのTime.nowはfreeze_timeで固定された値になる
	end
end
```

`before { freeze_time }` のように書くことで、各テストの前に時刻が固定される。
