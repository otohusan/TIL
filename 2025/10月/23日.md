## 途中のコミットを後から編集したかった
```
git rebase --interactive HEAD~~特定のコミットまで
```

修正したいコミットを **edit** に変更、**:wq** で保存する

この状態で適応したい修正をステージにあげて、以下を実行
```
git commit --amend --no-edit
```

修正が完了したら以下をする
```
git rebase --continue
```
